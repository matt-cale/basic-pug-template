## What is this?

This repo is used for demonstrating how to use pug from the command line.

## How do I run it?

1. `git clone ${this_repo}`
2. `cd ${this_repo}`
3. `npm i`
  * `npm run pugit`
  * or
  * `node_modules/.bin/pug index.pug --pretty`
